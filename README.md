# bubble-store

This is a Front End project that I did at Mindera School. <br/>
It is an e-commerce that sells FUNKO POP from Naruto and Fairy Tail. <br/>
You can add items to the cart. <br/>
Filter and sort products by price. <br/>
Search for products.
